wordpress实现同步到facebook、twitter、buzz等多个社交网站

2011-05-02 · 3shi

linksalpha 是个具有可以同步到多个社交网站的工具，它有收费的也有可以免费的，不过当然是选免费的咯，免费的可以连接facebook ,twitter,linkedin,myspace ,window live等等社交网站。可以增加5个社交网站的链接，5个发表文章的源头网站blog，还有其中一个网站可以拥有分析网站共享同步数据的功能，每天同步的文章可以高达300篇。一般网站已经足够了。 使用LinksAlpha来同步文章有两种方式：

第一种方式，使用LinksAlpha插件，就是在wordpress增加插件；

你可以在wordpress的后台"新增插件".然后搜索LinksAlpha ，然后会出现两个可以具有同步功能的插件： “Social Discussion” ：wordpress官方下载http://wordpress.org/extend/plugins/social-discussions/ “network publisher：wordpress官方下载 http://wordpress.org/extend/plugins/network-publisher/ 这两个都可以使用 LinksAlpha达到同步，区别在于，前者还可以设置社交网络共享按钮，比较丰富一点。

类似功能的插件还有：1-click Retweet/Share/Like wordpress官方下载http://wordpress.org/extend/plugins/1-click-retweetsharelike/

Network Publisher插件的使用：

Network Publisher 为 linksalpha 在 WordPress 上所作的plugin，提供了许多目前热门的 社群网络 的自动发布讯息的服务

1. WordPress 安装 Network Publisher plugin，并启动。 启动后，在 “插件” 中，会出现 Network Publisher 设置页面

2. 登陆 linksalpha 帐号 (可使用google, Facebook, twitter 帐号) 选择想要发布的 社群网络 服务，服务右方如果有 “$” 符号，代表是要付费才能使用 (ex:Plurk…)。免费用户，可以设置 5 个网站的讯息 至 5 个免费的社群网络来发布讯息。

3. 设置 linksalpha API KEY 至 WordPress 得到你的 linksalpha API KEY: http://www.linksalpha.com/user/your_api_key，设置到 WordPress Network Publisher 设置页面，之后就会在 Currently Publishing 中看到你想发布的社群网络列表。

第二，不用使用插件，直接在网站设置http://www.linksalpha.com，

然后增加你的网站，还有各个社交网络即可，连API key 都不用。方便多了，而且还可以减少对wordpress的负担，少了一个插件！

方法如下：

1 打开主页： http://www.linksalpha.com

2 先进入注册/登陆界面：http://www.linksalpha.com/auth/login?display=signup 登陆帐号之后，你可以授权给linksalpha.com linksalpha就可以帮助你主动发布文件。

不过，这个授权权限比较高，DaoFa.net提醒大家使用时自己把握用不用了。
