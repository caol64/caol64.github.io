不登录浏览需要登录才能浏览的论坛页面

2011-01-31 · 3shi

注意：本方法不能保证对所有的论坛都有效，但是操作很方便，大家可以尝试一下。

经常会碰到一些论坛，在浏览某些版面帖子的时候会遇到如下的提示：

您无权进行当前操作，这可能因以下原因之一造成
您所在的用户组(xxx)无法进行此操作。

这时，如果你不登录就无法浏览该页内容，而如果你不是注册用户的话就需要先注册，很是麻烦。 现在就给各位介绍一种破解该限制的方法。 先说原理：

虽然论坛限制了普通浏览器浏览，但它一般不会拒绝搜索引擎收录内容（搜索引擎可以带来流量嘛）。因此，我们要做的就是把我们的浏览器伪装成搜索引擎，骗过论坛程序。

知道了原理，就开始吧： 选择一个可以切换user-agent的浏览器，这里以firefox为例。 第一步：为firefox添加一个User Agent Switcher的扩展。你可以到它的官方网站：这里

直接选择“添加到Firefox”，它会自动安装，安装完后按照提示重启firefox。

第二步：找到无法浏览的那个页面，点击火狐的“工具”菜单，如下图，选择Default User Agent - Search Robots - Googlebot 2.1。

刷新页面，哈哈，是不是隐藏的内容出来了？ 操作完成后不要忘了切换回Default User Agent。
